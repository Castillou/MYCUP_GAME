# MYCUP_GAME

투표 게임 웹 앱 "MYCUP"

## 프로젝트 소개

이상형 월드컵 사이트에서 영감을 받아 만든 투표 커뮤니티 웹 앱입니다.
사용자는 두 가지 후보를 정해 투표를 시작할 수 있고,
투표에 참여한 유저들은 투표를 진행하고 현황을 확인하며 투표에 참여한 다른 참가자들과 댓글을 통해 소통할 수 있습니다.
